# 🚀 PETI-Generator: Generador de Plan Estratégico de TI

Una **aplicación web interactiva** diseñada para guiar a los usuarios en la creación, gestión y consolidación de un **Plan Estratégico de Tecnologías de la Información (PETI)** de manera estructurada y profesional.

Este proyecto fue desarrollado como una **herramienta académica y de gestión** para estandarizar el proceso de planificación estratégica, permitiendo a los usuarios definir desde la *misión y visión* de su empresa hasta realizar análisis complejos como **FODA, PEST y CAME**.

---

## ✨ Características Principales

- 🔐 **Autenticación de Usuarios**: Sistema seguro de registro e inicio de sesión.  
- 📂 **Gestión de Múltiples Proyectos**: Un solo usuario puede crear y gestionar planes para distintas empresas o proyectos.  
- 🧭 **Interfaz Guiada**: Un *dashboard* interactivo que lleva al usuario paso a paso a través de los **11 módulos del plan estratégico**.  
- 📝 **Módulos de Información General**: Secciones dedicadas para definir *Misión, Visión, Valores y Objetivos*.  
- 📊 **Módulos de Análisis Estratégico**: Herramientas para completar análisis clave:  
  - **FODA** (*Fortalezas, Oportunidades, Debilidades, Amenazas*)  
  - **PEST** (*Político, Económico, Social, Tecnológico*)  
  - **CAME** (*Corregir, Afrontar, Mantener, Explotar*)  
  - **Cadena de Valor de Porter**  
  - **Matriz BCG** y más.  
- 💾 **Persistencia de Datos**: Toda la información se guarda en tiempo real en la base de datos, permitiendo continuar el trabajo en cualquier momento.  
- 🎨 **Diseño Moderno y Responsivo**: Interfaz oscura con acentos de color, construida con *Bootstrap 5* para una experiencia fluida en cualquier dispositivo.  

---

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido con las siguientes tecnologías:

- **Backend**: PHP 8.2  
- **Base de Datos**: MySQL (gestionado localmente con *XAMPP* y en la nube con *Google Cloud SQL*).  
- **Frontend**: HTML5, CSS3, Bootstrap 5  
- **Entorno de Desarrollo Local**: XAMPP (*Apache, MySQL, PHP*)  
- **Plataforma de Despliegue**: Google Cloud Platform (*App Engine* para la aplicación y *Cloud SQL* para la base de datos).
